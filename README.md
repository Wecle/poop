# 噗噗 - React Native 健康习惯追踪应用

"噗噗"是一款使用 React Native（Expo）构建的健康习惯追踪应用，能够在 iOS 与 Android 端运行。应用帮助用户记录、统计如厕时长，让培养规律生活习惯变得更轻松。

## 项目背景

良好的排便习惯是健康生活的重要组成部分。通过记录和分析如厕数据，用户可以：
- 了解自己的身体规律
- 发现潜在的健康问题
- 培养规律的生活习惯
- 用轻松有趣的方式关注健康

## 主要功能

### 🏠 首页
- 一键开始/结束计时，自动记录时长
- 展示最近一次记录的完成时间与持续时间
- 友好的圆形倒计时按钮，提醒用户当前状态

### 📅 日历
- 按日期分组查看所有历史记录
- 清晰展示每天完成的次数与时长
- 空数据时提供指引文案

### 📊 统计
- 按「今日 / 本周 / 本月」统计次数与时长
- 计算平均时长，帮助了解排便习惯
- 提供本月活跃日概览

### 👤 我的
- 支持昵称编辑与主题色选择
- 可配置提醒、震动、提示音等偏好
- 设置保存在本地，随时加载

## 本地开发

```bash
pnpm install
pnpm start
```

使用 Expo Go 扫描二维码即可在真机上预览，或运行 `pnpm android` / `pnpm ios` 在模拟器中调试。

## 技术栈

- [Expo](https://expo.dev/) SDK 54
- React Native 0.81
- AsyncStorage 持久化数据
- date-fns 进行时间计算
